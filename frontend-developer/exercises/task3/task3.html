Task 3 - Tag Form (JS)

For this exercise you need to build a simple tag form - a form that manages a list of unique values.

You might have seen more complex versions of this before. Here are a few examples:l

Requirements

We just want to implement the basic functionality:

  • An input to add a tag to the list
    • If the tag is already in the list, it should not add it
    • The input's value should be reset if the value is added
  • A list of all the added tags.
    • Each item should have a button to remove itself from the list.
