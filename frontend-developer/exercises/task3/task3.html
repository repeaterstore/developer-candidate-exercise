Task 3 - Remote Data (JS)

For this exercise you need to:

  • Take the data from the form input and send it to the server.
  • Render the message from the response in the "status-message" div element.

Getting started

  • Install dependencies by running npm install
  • Start the web server by running npm run start. The server will run on port 3000
  • Make sure you're loading this page from http://localhost:3000 so you don't run into CORS issues

Endpoint

POST /message

Request

Request body must be in JSON

Example: { "name": "Ada" }

Response

Response is sent as JSON

HTTP Status Code Description Example
200 Success { "message": "Thanks Ada" }
400 Validation failed { "error": "Name must not be blank" }
